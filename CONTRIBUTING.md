# Contributing to FlowPilot AI

Thank you for showing interest in helping build and optimize FlowPilot AI! As an open-source Series-A platform, we welcome pull-requests, feedback issues, and testing compliance checks.

## Development Workflow

1. Fork the repository and check out a dedicated feature branch:
   ```bash
   git checkout -b feature/gorgeous-orbs
   ```
2. Verify typing validations and style lints:
   ```bash
   npm run lint
   ```
3. Run the automated compliance test runner before pushing code:
   ```bash
   npm run test
   ```
4. Submit your pull-request outlining visual modifications and API summaries!
