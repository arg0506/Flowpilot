# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| v1.0.x  | Yes       |

## Reporting a Vulnerability

We take the security of our platform, private keys, and user secrets extremely seriously. If you identify an issue (such as client-side key leakage, RPC gateways exploits, or bypass vulnerabilities), please do not open a public GitHub issue.

Directly dispatch a detailed write-up to our security team: **argha0506@gmail.com**.

We pledge to review and respond to reports within 48 hours and formulate release patches immediately. Thank you for helping keep FlowPilot AI secure!
